## 输出格式
必须是严格的JSON格式，使用代码块包裹：

- 必须以 ```json 开头，以 ``` 结尾
- 所有字段值中严禁出现未转义的双引号，改用单引号或转义为 \"

```json
{
  "claims": [
    {
      "subject": "",
      "predicate": "",
      "object": "",
      "origin": "",
      "time": "",
      "content": "",
      "source": "",
      "confidence": 0.0,
      "tags": []
    }
  ]
}
```

**格式规范**：
- 每条 claim 只能出现一次，禁止重复
- 每个字段值必须在同一行，不要在字符串中间换行
- 每个字段都必须输出，不能省略
- 无值的 time 填"未知"
